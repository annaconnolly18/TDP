Data Engineering¶

We can see that we can join both datasets by the date, but there are 3 present problems.
1.) The datasets are grouped by different values. Weather dataset has monthly values, and murder dataset has daily values.
2.) The datasets have different date ranges
3.) We have to un-pivot the weather dataset in order to be able to join with the murder dataset

Group discrepencies and Date Ranges¶

We'll begin by solving the first two problems by grouping the murder dataset to a monthly view. Then we'll set the date range to only be after 2014. We'll calculate the total number of murders per month
